The code is an automatic extraction algorithm for shallow water areas in remote sensing images based on C++ programming.
The programming platform is Visual Studio 2013, and the external dependency library is GDAL.
The test file contains the algorithm's executable program seaLandSplit.exe and several dependent dynamic libraries. 
Double-clicking the seaLandSplit.exe file will run the program directly. 
testImage.tiff is the test data.
